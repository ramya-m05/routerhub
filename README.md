# Routerkart

Routerkart is a full-stack inventory management system for router products.

## Tech Stack

Frontend:
- React (Vite)
- Axios

Backend:
- Node.js
- Express
- MongoDB

## Features

- Admin login (JWT authentication)
- Add router products
- Edit router details
- Delete routers
- View router inventory

## Setup

Backend:

npm install  
node server.js

Frontend:

cd frontend  
npm install  
npm run dev
